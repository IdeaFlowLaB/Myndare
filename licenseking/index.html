Features

四個功能，把證照管得服服貼貼

📸

拍照即管理

用相機掃描證件，OCR 自動辨識姓名、證號、有效期限與發證機構。整個過程在裝置本機完成，照片不會送到任何雲端。

📱

手機就是員工證

掃描實體員工證或會員卡的條碼，App 會在手機上重新生成可掃描的條碼。搭配 WidgetKit 桌面小工具，需要時一秒拿出來。

⏰

到期不再忘

智慧提醒在到期前 30 天、7 天、1 天自動推播通知。提醒時間完全可自訂，讓你從容準備續證，不會錯過任何續證時機。

☁️

換機不怕

雲端同步備份（付費方案），換手機或多裝置都能無縫接續。資料以 HTTPS 加密傳輸，受 Row Level Security 保護，僅本人可存取。
