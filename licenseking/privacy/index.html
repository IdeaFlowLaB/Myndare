Privacy Policy

隱私權政策

最後更新日期：2026 年 4 月 8 日

Myndare（以下簡稱「我們」）尊重您的隱私。本政策說明證照王 License King（以下簡稱「本 App」）如何蒐集、使用、儲存與保護您的資料。使用本 App 即代表您同意本政策的內容。

01我們收集的資料

本 App 蒐集的所有資料皆為您主動提供或自願輸入。本 App 不會在背景自動蒐集任何資料。

帳號資訊

  • 電子郵件地址（用於註冊與登入付費方案；免費版可僅本機使用，無需註冊）

證照與證件資訊

  • 證照名稱、持證人姓名、證件號碼、有效日期、發證機構等欄位資料
  • 您主動拍攝或從相簿選取的證件照片

條碼資料

  • 條碼原始字串與類型（QR Code / Code 128 / EAN 等），僅用於本 App 的「員工證重新生成」功能

個人資訊（選填）

  • 履歷卡功能中可選填的姓名、求職方向、自我介紹等

本 App 不蒐集以下資料：廣告識別碼（IDFA）、地理位置、通訊錄、瀏覽記錄、裝置使用行為、其他 App 的使用資訊。本 App 不使用任何分析或追蹤 SDK，包括但不限於 Firebase Analytics、Crashlytics、Facebook SDK、Google Analytics 等第三方分析工具。

02資料使用目的

我們僅將您的資料用於以下目的：

  • 提供證照管理核心功能：證照建檔、OCR 辨識、條碼重新生成、到期提醒
  • 雲端同步與備份（僅付費方案）
  • 家庭共享功能（僅家庭方案）
  • 應您要求提供客服支援

我們不會將您的資料出售、出租或交換給任何第三方，也不會用於廣告投放或行為分析。

03資料儲存與安全

本機儲存

所有資料預設儲存於您的 iOS 裝置本機，使用 iOS 內建加密與 Core Data 的 Complete File Protection，僅在裝置解鎖時可存取。

雲端儲存（僅付費方案）

啟用雲端同步後，資料會以 HTTPS / TLS 加密傳輸至 Supabase（基礎設施由 AWS 提供，伺服器位於亞太區）。每位使用者的資料皆受 Row Level Security（RLS）規則保護，僅本人可存取。

「僅儲存於本機」選項

付費使用者可隨時在設定中關閉雲端同步，所有資料將僅保留於本機，不會上傳至雲端。

04OCR 技術

本 App 使用 Google ML Kit 的 on-device OCR 技術進行證照文字辨識。整個辨識過程完全在您的裝置本機完成，照片與辨識結果不會傳送至 Google 或任何第三方伺服器。

05第三方服務

本 App 使用以下第三方服務，並僅在必要範圍內傳輸資料：

  • Supabase（雲端儲存與認證）：僅在啟用雲端同步或使用付費方案時，傳輸您的證照資料與帳號資訊。
  • Apple Sign-In / Google Sign-In：僅在您選擇社群登入時，從 Apple 或 Google 接收必要的識別資訊（電子郵件、使用者識別碼）。
  • Google ML Kit（OCR）：僅在裝置端執行，不傳輸資料。
  • Apple StoreKit：處理付費方案的訂閱與付款，由 Apple 負責付款資料處理；本 App 不接觸您的信用卡資訊。

06使用者權利

您可以隨時行使以下權利：

  • 存取：在 App 內隨時查看您所有的證照資料
  • 匯出：將證照資料匯出為檔案備份
  • 修改：編輯任何欄位內容
  • 刪除：刪除單一證照或整個資料庫
  • 刪除帳號：在設定頁面底部點選「刪除帳號」，30 天內所有雲端資料將永久清除
  • 停止同步：隨時關閉雲端同步功能

07兒童隱私

本 App 不針對 13 歲以下兒童設計，也不會主動向兒童蒐集個人資料。如您發現未滿 13 歲的兒童在未經監護人同意下使用本 App，請來信告知，我們將儘速移除相關資料。

08資料保留

  • 本機資料：保留至您主動刪除或從裝置移除本 App 為止
  • 雲端資料：保留至您刪除帳號為止；帳號刪除後，所有雲端資料將於 30 天內永久清除

09政策變更

當本政策有重大變更時，我們會在 App 內通知您。建議您定期查看本頁以了解最新版本。繼續使用本 App 即代表您接受變更後的政策內容。
